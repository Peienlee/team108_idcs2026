# IDSC 2026: Hope for Glaucoma Detection

## 👁️ Vision
Early detection of glaucoma prevents blindness. This project brings hope by building an interpretable AI model for glaucoma screening using retinal fundus images.

## 📂 Dataset
- **HYGD (Hillel Yaffe Glaucoma Dataset)** from PhysioNet
- 1,300+ fundus images with expert-labeled glaucoma diagnosis
- Includes image quality scores
